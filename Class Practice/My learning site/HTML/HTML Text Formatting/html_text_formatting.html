HTML Text Formatting

HTML contains several elements for defining text with a special meaning.

HTML Formatting Elements

  • <b> - Bold text
  • <strong> - Important text
  • <i> - Italic text
  • <em> - Emphasized text
  • <mark> - Marked/Highlight text
  • <small> - Smaller text
  • <del> - Deleted text
  • <ins> - Inserted/Underlined text
  • <sub> - Subscript text
  • <sup> - Superscript text
  • <blockquote> - Blockquote/Big Quotation text
  • <q> - Short Quotation text
  • <abbr> - Abbreviation text
  • <address> - Contact/Owner Information
  • <cite> - title of a creative work (e.g. a book, a poem, a song, a movie, a painting, a sculpture, etc.).
  • <bdo> - Bi-Directional text
